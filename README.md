# Emotion Detector

An AI-based web application that detects emotions from text using the Watson NLP library.

## Project Name
Emotion Detector - Ghadi Alghamdi

## Features
- Emotion analysis using Watson NLP API
- Web interface built with Flask
- Error handling for blank/invalid input
- Unit tested with Python unittest
- Static code analysis with PyLint (score: 10/10)
